POST /katello/api/content_view_versions/incremental_update
Realizar una actualización incremental en uno o más versiones de vista de contenido

Parámetros

Nombre del parámetro Descripción
content_view_version_environments
opcional

Validations:

  • Must be an Array of nested elements
content_view_version_environments[content_view_version_id]
opcional

ID de versión de vista de contenido para realizar una actualización incremental. Puede contener compuestos al igual que uno o más componentes para actualizar.

Validations:

  • string from 2 to 128 characters containing only alphanumeric characters, space, ‘‘, ‘-’ with no leading or trailing space..
content_view_version_environments[environment_ids]
opcional

La lista de entornos para promover la versión de vista de contenido (remplaza la versión anterior)

Validations:

  • Must be an array of any type
description
opcional

La descripción para las nuevas versiones de contenido generadas

Validations:

  • String
resolve_dependencies
opcional

Si es verdadero, cuando agregue las erratas o paquetes especificadas, cualquier dependencia que se requiera, también se copiará.

Validations:

  • boolean
propagate_all_composites
opcional

Si es verdadero, publicará una nueva versión compuesta mediante content_view_version_id que ha sido promovido a un entorno de ciclo de vida.

Validations:

  • boolean
add_content
opcional

Validations:

  • Hash
add_content[errata_ids]
opcional

Errata uuids to copy into the new versions.

Validations:

  • Must be an array of any type
add_content[package_ids]
opcional

Package uuids to copy into the new versions.

Validations:

  • Must be an array of any type
add_content[puppet_module_ids]
opcional

Puppet Modules to copy into the new versions.

Validations:

  • Must be an array of any type
update_systems
opcional

Después de generar la actualización incremental, aplique los cambios a los sistemas especificados. Solo las erratas reciben soporte actualmente.

Validations:

  • Hash
update_systems[included]
requerido

Validations:

  • Hash
update_systems[included][search]
opcional , nil permitido

Buscar cadena para sistemas en los que realizar una acción

Validations:

  • String
update_systems[included][ids]
opcional , nil permitido

Lista de ID de sistema sobre los que realizar una acción

Validations:

  • Must be an array of any type
update_systems[excluded]
opcional , nil permitido

Validations:

  • Hash
update_systems[excluded][ids]
opcional , nil permitido

Lista de ID de sistema para excluir y no ejecutar ninguna acción en ellos

Validations:

  • Must be an array of any type
update_systems[update_all_systems]
opcional

Actualizar todos los sistemas modificables y aplicables, no solo los que usan las versiones de vista de contenido y entornos seleccionados

Validations:

  • boolean
