POST /katello/api/content_view_versions/:id/export
Export a content view version

Params

Param Name Description
id
Required

Content view version identifier

Validations:

  • string from 2 to 128 characters containing only alphanumeric characters, space, ‘‘, ‘-’ with no leading or trailing space..
export_to_iso
Optional

Export to ISO format

Validations:

  • boolean
iso_mb_size
Optional

maximum size of each ISO in MB

Validations:

  • number.
since
Optional

Optional date of last export (ex: 2010-01-01T12:00:00Z)

Validations:

  • Must be a Date in the form of YYYY-MM-DD.
