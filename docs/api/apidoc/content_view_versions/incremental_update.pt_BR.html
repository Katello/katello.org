POST /katello/api/content_view_versions/incremental_update
Realizar uma Atualização Adicional em um ou mais Versões de Visualização de Conteúdo

Parâmentros

Nome parâmetro Descrição
content_view_version_environments
opcional

Validations:

  • Must be an Array of nested elements
content_view_version_environments[content_view_version_id]
opcional

Ids da Versão da Visualização de Conteúdo onde realizar uma atualização adicional. Pode conter composição igual ou com mais componentes para atualização.

Validations:

  • string from 2 to 128 characters containing only alphanumeric characters, space, ‘‘, ‘-’ with no leading or trailing space..
content_view_version_environments[environment_ids]
opcional

A lista de ambientes para promover a Versão de Visualização do conteúdo especificado para (substituindo a versão mais antiga).

Validations:

  • Must be an array of any type
description
opcional

A descrição para nova versão da visualização de conteúdo gerada

Validations:

  • String
resolve_dependencies
opcional

Se verdadeiro, ao adicionar a errata ou pacotes especificados, qualquer dependência será copiada também.

Validations:

  • boolean
propagate_all_composites
opcional

Se verdadeiro, será publicado uma nova versão de composição utilizando qualquer content_view_version_id especificado que já tenha sido promovido para um ambiente de ciclo de vida.

Validations:

  • boolean
add_content
opcional

Validations:

  • Hash
add_content[errata_ids]
opcional

Errata uuids to copy into the new versions.

Validations:

  • Must be an array of any type
add_content[package_ids]
opcional

Package uuids to copy into the new versions.

Validations:

  • Must be an array of any type
add_content[puppet_module_ids]
opcional

Puppet Modules to copy into the new versions.

Validations:

  • Must be an array of any type
update_systems
opcional

Após gerar a atualização adicional, aplicar as mudanças em sistemas especificados. Atualmente, somente erratas são suportadas.

Validations:

  • Hash
update_systems[included]
requerido

Validations:

  • Hash
update_systems[included][search]
opcional , nulo permitido

Buscar faixa para sistemas a realizar uma ação em

Validations:

  • String
update_systems[included][ids]
opcional , nulo permitido

Lista de IDs de sistema na qual executar uma ação

Validations:

  • Must be an array of any type
update_systems[excluded]
opcional , nulo permitido

Validations:

  • Hash
update_systems[excluded][ids]
opcional , nulo permitido

Lista de ids de sistema para excluir e não executar uma ação

Validations:

  • Must be an array of any type
update_systems[update_all_systems]
opcional

Atualize todos os sistemas editáveis e aplicáveis, e não apenas aqueles que utilizam as Versões e Ambientes de Visualizações de Conteúdo selecionadas.

Validations:

  • boolean
