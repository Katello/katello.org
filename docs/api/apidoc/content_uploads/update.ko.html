PUT /katello/api/repositories/:repository_id/content_uploads/:id
Upload a chunk of the file's content

Params

Param Name Description
repository_id
Required

Repository id


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
id
Required

Upload request id


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
offset
Required

The offset in the file where the content starts


Value: number.
content
Required

The actual file contents


Value: File
