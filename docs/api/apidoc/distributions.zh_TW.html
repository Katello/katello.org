Distributions

>>>

GET /katello/api/repositories/:repository_id/distributions
List distributions

Params

Param Name Description
repository_id
Optional

Repository id to list packages for


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
search
Optional

Search string


Value: String
page
Optional

Page number, starting at 1


Value: number.
per_page
Optional

Number of results per page to return


Value: number.
order
Optional

Sort field and order, eg. ‘name DESC’


Value: String
full_results
Optional

Whether or not to show all results


Value: boolean
sort
Optional

Hash version of ‘order’ param


Value: Hash
sort[by]
Optional

Field to sort the results on


Value: String
sort[order]
Optional

How to order the sorted results (e.g. ASC for ascending)


Value: String
