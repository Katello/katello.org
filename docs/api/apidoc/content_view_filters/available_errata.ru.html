GET /katello/api/content_views/:content_view_id/filters/:id/available_errata
Get errata that are available to be added to the filter

GET /katello/api/content_view_filters/:id/available_errata
Get errata that are available to be added to the filter

Params

Param Name Description
content_view_id
Optional

content view identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
id
Required

filter identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
types
Optional

Errata types array [‘security’, ‘bugfix’, ‘enhancement’]


Value: Must be an array of any type
start_date
Optional

Start date that Errata was issued on to filter by


Value: DateTime
end_date
Optional

End date that Errata was issued on to filter by


Value: DateTime
