Export a repository

Params

Param name Description
id
required

Repository identifier

Validations:

export_to_iso
optional

Export to ISO format

Validations:

  • boolean
iso_mb_size
optional

maximum size of each ISO in MB

Validations:

  • number.
since
optional

Optional date of last export (ex: 2010-01-01T12:00:00Z)

Validations:

  • Must be a Date in the form of YYYY-MM-DD.
